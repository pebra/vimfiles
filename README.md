## vimfiles ##

### Installation ###

```sh
  git clone https://github.com/pebra/vimfiles.git ~/.vim

  mkdir -p ~/.vim/bundle

  git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

  ln -s ~/.vim/vimrc ~/.vimrc
```

open up vim and then run

```sh
  :BundleInstall
```

now you're ready to go
